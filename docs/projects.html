Rhino

Timeline: 2018 - 2020
Website
Location: New York, USA
Industry: FinTech
Rhino is a real estate technology company replacing security deposits with a smarter alternative that’s better for both renters and landlords. Since 2018 we have been working on this project in close cooperation with the client developers team in NY. Our engineers work as Ruby on Rails and React.js/Redux/TypeScript developers. Significant achievements: built MVP almost from scratch; added integration with Yardi Voyager, developed recurring subscriptions flow with multiple Stripe accounts; built complex automatic reporting system for insurance partners.

Form2chat.io

Timeline: 2019 - 2020
Website
Location: Tallin, Estonia
Industry: Marketing & Advertising

Form2chat is a serverless form backend that allows to receive form submissions directly to major chat platforms: Telegram, Messenger, Slack and Viber without using email or any additional services. Form2chat also provides advanced features like spam protection, auto-responses with custom templates, file uploads and Zapier integration. A team of developers from our company implemented this project from scratch and provided a full development service from initial design planning to maintenance in production.

Team: 2 full stack developers, 1 frontend developer
The scope of our work: Full development cycle
Technologies and services we've used:

Frontend: jQuery
Integrations: Paddle.comTelegramViberAmazon Simple Email Service

BlackInk

Timeline: 2019
Website
Location: Paris, France
Industry: FinTech, Media

BlackInk project goal was to simplify paywall integration for news and magazine publishers and provide an all-in-one platform both for readers and publishers. Our team built this project from scratch, and it consisted of 3 separate services:

  • Analytics API to track customers behavior through payment flow (Elixir and PhoenixFramework)
  • Readers dashboard with purchases and wallet (Ruby on Rails)
  • Lightweight javascript paywall library to be embedded to publishers websites
